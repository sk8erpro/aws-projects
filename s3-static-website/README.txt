# S3 Static Website Hosting - Mini Jump Game

This project demonstrates hosting a **simple mini game** on **AWS S3**.  
The goal is to showcase practical use of AWS services for deployment while including a small interactive project.

## Project Description
- A small **jumping game** where the player navigates a character across platforms.
- Built with **HTML, CSS, and JavaScript**.
- Hosted on **AWS S3** as a static website.
- Demonstrates how to configure S3 bucket for static website hosting.

## AWS Services Used
- **S3**: To store and host the game files.

## Files
- `index.html` → main game page  
- `style.css` → styling of the game  
- `script.js` → game logic and interactions
